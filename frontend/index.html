🛠️ Технологии

Backend: FastAPI, Python 3.14
Database: PostgreSQL, asyncpg
Auth: JWT, argon2
ORM: SQLAlchemy 2.0 (async)
Migrations: Alembic
Container: Docker, Docker Compose

📡 API Эндпоинты

🔐 Аутентификация

POST /api/auth/register Регистрация нового пользователя
Body: {"username": "string", "password": "string", "email": "string"}
POST /api/auth/login Вход в систему
Body: {"username": "string", "password": "string"} → {"access_token": "jwt", "token_type": "bearer"}
GET /api/auth/me Получить профиль (требуется токен)
Header: Authorization: Bearer <token>

📖 Книги

POST /api/book/ Создать книгу (требуется токен)
Body: {"title": "string", "genre": "string", "description": "string"}
GET /api/book/ Список книг (пагинация, фильтры)
Параметры: ?limit=50&page=1&genre=fantasy&author_id=1
GET /api/book/{id} Получить книгу по ID
PATCH /api/book/{id} Обновить книгу (только владелец)
DELETE /api/book/{id} Удалить книгу (только владелец)

📝 Примеры запросов (curl)

Регистрация

curl -X POST http://localhost:8000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username": "john", "password": "secret123", "email": "john@example.com"}'

Логин

curl -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "john", "password": "secret123"}'

Создание книги (требуется токен)

curl -X POST http://localhost:8000/api/book/ \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <ваш_токен>" \
  -d '{"title": "1984", "genre": "drama", "description": "Anti-utopia"}'

Список книг с фильтрацией

curl "http://localhost:8000/api/book/?limit=10&page=1&genre=fantasy"

⚠️ Коды ответов

200OK - Успешно
201Created - Создано
400Bad Request - Неверный запрос
401Unauthorized - Требуется токен
403Forbidden - Недостаточно прав
404Not Found - Не найдено
409Conflict - Конфликт (дубликат)
422Unprocessable - Ошибка валидации
503Service Unavailable - БД недоступна
